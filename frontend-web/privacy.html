Seguridad y datos

Política de privacidad

CommerCity protege los datos de usuario y conserva controles de cuenta, notificaciones y privacidad.

Datos de cuenta

Se usan para autenticación, roles y operación de pedidos, tiendas y soporte.

Preferencias

Las preferencias visuales y de sesión se guardan de forma local o en backend según el módulo.

Solicitudes

El usuario puede solicitar desactivación o eliminación lógica según las reglas del sistema.
